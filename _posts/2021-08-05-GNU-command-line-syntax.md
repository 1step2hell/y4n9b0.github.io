如下的扩展方法应该是大家都很容易写出来的：

```kotlin
fun View.visible(): View = apply { visibility = View.VISIBLE }

fun View.invisible(): View = apply { visibility = View.INVISIBLE }

fun View.gone(): View = apply { visibility = View.GONE }
```
